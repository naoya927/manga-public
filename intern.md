# 漫画感想記録Webアプリについて

このアプリは、生成AIの発展に伴って注目されているバイブコーディングを通して、実際に自分の手でWebアプリを作る体験をしてみたいという思いから開発しました。もともと漫画が好きで、読んだ作品の感想や印象に残った名言、自分なりの解釈を後から振り返れる形でまとめたいと考えていたため、漫画の感想記録に特化したアプリとして設計しました。最初はローカル環境で動く `v3` として、HTML/CSS/JavaScript による画面と Python の簡易サーバーを組み合わせ、感想を Markdown、読書履歴を JSON、画像をローカルファイルとして保存する構成から始めました。その後、より実用的に使える形にするため、`my-supabase-app` として Supabase と Vercel を利用した構成へ発展させ、感想データを Supabase Postgres、画像を Supabase Storage、API を Vercel Functions、フロントエンドを Vercel の静的配信で扱うようにしました。アプリでは、漫画タイトル、よみがな、名言、感情タグ、感想、箇条書き要約、表紙画像、ギャラリー画像、読書カレンダー、お気に入り状態などを記録できます。また、OpenAI API を利用してタイトルのよみがな生成や感想の箇条書き要約も行えるようにしています。使用技術は、フロントエンドが HTML/CSS/JavaScript、バックエンドが Python の `http.server` と Node.js/Vercel Functions、データベースが Supabase Postgres、画像管理が Supabase Storage、デプロイが Vercel、既存データ移行が TypeScript によるスクリプトです。個人の感想や画像データは GitHub には載せず、コードとアプリ構造を公開する方針にすることで、開発過程や技術的な工夫を見せながら、個人情報に近い記録は保護できる構成にしています。
